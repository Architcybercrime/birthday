# Happy Birthday Anshika 🎉

A Happy Birthday animation built with CSS3, HTML5 & jQuery.

Made with ❤️ by **Archit Agrawal**  
GitHub: [Architcybercrime](https://github.com/Architcybercrime)

## Tech Used
HTML5 · CSS3 · jQuery · Less CSS

## Run Locally

**With Python:**
```bash
python -m http.server 8081
```
Visit http://localhost:8081

**With Node.js:**
```bash
npm install
npm run server-node
```
Visit http://localhost:8081
